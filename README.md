## 📖 Overview
Welcome to **Nara AI Dashboard**, a cutting-edge platform built for professionals, researchers, and creators. By leveraging state-of-the-art Generative AI models, this studio streamlines the content creation pipeline, transforming raw prompts and data into fully structured, beautifully formatted presentations.

## ✨ Key Features
* **🧠 Intelligent AI Engine:** Powered by state-of-the-art models to ensure precise, deep, and structured content outlines.
* **⚡ Instant Export (AI-to-PPTX):** Seamlessly converts AI-generated content into fully formatted, ready-to-present PowerPoint files (16:9 ratio).
* **🎨 Corporate Standard Design:** Integrated visual system following the "Nara" branding identity. Features a professional **Navy Blue & Gold** palette with full RTL (Right-to-Left) support for high-quality Arabic presentations.
* **🎯 Structured Precision:** Optimized data matrices that organize slide titles, subtitles, and concise bullet points to deliver impactful and professional narratives.
* **📱 Fully Responsive UI:** Optimized for seamless performance across all devices, from desktop monitors to mobile screens, with dedicated code-block formatting.

## 🔑 API Configuration & Setup
To unlock the full generation capabilities of the dashboard, a valid API key is required.

### 1. Get Your API Key
1. Visit our API provider to create your account: [NaraRouter API](https://router.bynara.id/)
2. Upon registration, you are granted **7,000,000 free tokens** to utilize within the dashboard.
3. Generate your secure API key.

### 2. Configure the Dashboard
Navigate to the settings tab within the Nara AI Dashboard, paste your API key, and begin generating your professional presentations instantly.
